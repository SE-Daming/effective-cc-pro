# 业务规则说明

> 定义各业务模块的核心规则，作为开发和测试的依据

---

## 1. 选座锁座规则

| 规则项 | 说明 |
|--------|------|
| 锁座时长 | 选座后 **5 分钟** 内未支付，自动释放座位 |
| 并发选座 | 同一座位被多人同时选择时，先下单者锁定，后者提示"座位已被选" |
| 锁座状态 | 锁定期间座位对其他用户显示为"已选"状态 |
| 超时释放 | 超时未支付，座位自动变为可选状态，订单自动取消 |

---

## 2. 退票规则

| 规则项 | 说明 |
|--------|------|
| 可退票时间 | 开场前 **2 小时** 可申请退票 |
| 不可退票 | 开场前 2 小时内、已取票、已观影后不可退票 |
| 退票手续费 | 开场前 24 小时以上：无手续费 |
|          | 开场前 2-24 小时：收取票价 **10%** 手续费 |
| 退款到账 | 退款原路返回，1-3 个工作日到账 |
| 优惠券退还 | 若使用优惠券，优惠券不退回 |

---

## 3. 优惠券规则

### 3.1 优惠券类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 满减券 | 满足金额后减免 | 满 50 减 10 |
| 折扣券 | 指定折扣 | 票价 8 折 |
| 立减券 | 无门槛直接减免 | 立减 5 元 |
| 兑换券 | 免费兑换指定商品 | 免费兑换小份爆米花 |

### 3.2 使用规则

| 规则项 | 说明 |
|--------|------|
| 使用范围 | 可指定：全部电影、指定电影、指定影院 |
| 有效期 | 领取后 **7-30 天** 内有效（按券类型） |
| 叠加使用 | 每笔订单限用 **1 张** 优惠券 |
| 退款处理 | 使用优惠券的订单退款，优惠券不退回 |

---

## 4. 订单状态流转

### 4.1 电影票订单

```
待支付 → 已支付 → 已出票 → 已完成
   ↓        ↓
 已取消   退款中 → 已退款
```

| 状态 | 说明 |
|------|------|
| 待支付 | 订单创建，等待支付（5 分钟超时） |
| 已支付 | 支付成功，等待出票 |
| 已出票 | 生成取票码，可去影院取票 |
| 已完成 | 观影结束 |
| 已取消 | 超时未支付或用户主动取消 |
| 退款中 | 用户申请退款，等待审核/处理 |
| 已退款 | 退款成功 |

### 4.2 卖品订单

```
待支付 → 已支付 → 待取货 → 已完成
   ↓        ↓
 已取消   退款中 → 已退款
```

| 状态 | 说明 |
|------|------|
| 待取货 | 已支付，等待用户到影院取货 |
| 已完成 | 用户已取货 |

---

## 5. 支付规则

| 规则项 | 说明 |
|--------|------|
| 支付方式 | 微信支付 |
| 支付超时 | 订单创建后 **5 分钟** 内完成支付 |
| 支付金额 | 订单金额 = 票价 × 数量 + 卖品金额 - 优惠券 |
| 最小支付金额 | 订单最低支付 **0.01 元** |
