# AI 助手模块需求说明

> 独立 Spring Boot 服务，提供智能客服、智能推荐、Function Call 能力

---

## 1. 模块概述

### 1.1 核心定位

综合智能助手，具备三大核心能力：

| 能力 | 说明 |
|------|------|
| **智能客服** | 业务咨询、规则解释 |
| **智能推荐** | 电影、影院、优惠券推荐 |
| **Function Call** | 意图识别后调用业务接口完成操作 |

### 1.2 技术方案

| 技术选型 | 说明 |
|----------|------|
| Spring AI | AI 应用开发框架 |
| 百炼平台 API | 阿里云大模型平台 |
| HTTP | 调用主服务接口获取数据 |

### 1.3 服务架构

```
┌─────────────────────────────────────────────────────┐
│                    前端层                            │
│         ┌─────────────┐    ┌─────────────┐         │
│         │  Vue Web    │    │  UniApp     │         │
│         └──────┬──────┘    └──────┬──────┘         │
└────────────────┼──────────────────┼────────────────┘
                 │ HTTP             │ HTTP
                 ▼                  ▼
┌─────────────────────────────────────────────────────┐
│                   后端层                             │
│         ┌─────────────────────────────────┐         │
│         │      Spring Boot 主服务          │         │
│         │      (业务逻辑)                  │         │
│         └──────────────▲──────────────────┘         │
│                        │ HTTP                       │
│                        │                            │
│         ┌──────────────┴──────────────────┐         │
│         │    Spring Boot AI 助手模块       │         │
│         │      (智能对话 + 推荐调度)        │         │
│         └──────────────┬──────────────────┘         │
│                        │ API                         │
│                        ▼                            │
│         ┌─────────────────────────────────┐         │
│         │       百炼平台 API               │         │
│         │      (大模型能力)                │         │
│         └─────────────────────────────────┘         │
└─────────────────────────────────────────────────────┘
```

## 2. 智能客服能力

### 2.1 业务咨询

| 场景 | 示例问题 | AI 回答 |
|------|----------|---------|
| 电影查询 | "最近有什么科幻片上映？" | 列出当前热映的科幻电影 |
| 影院查询 | "附近有什么电影院？" | 列出附近影院及距离 |
| 排片查询 | "万达影院今天有什么电影？" | 列出该影院今日排片 |
| 票价查询 | "《流浪地球3》票价多少？" | 返回票价信息 |

### 2.2 规则解释

| 场景 | 示例问题 | AI 回答 |
|------|----------|---------|
| 退票规则 | "电影票怎么退？" | 解释退票时间限制、手续费规则 |
| 积分规则 | "积分怎么用？" | 解释积分获取、使用规则 |
| 会员规则 | "黄金会员有什么权益？" | 列出黄金会员权益 |
| 优惠券规则 | "这个券怎么用？" | 解释优惠券使用条件、有效期 |

## 3. 智能推荐能力

### 3.1 电影推荐

| 推荐场景 | 推荐逻辑 | 数据来源 |
|----------|----------|----------|
| 基于偏好 | 根据用户观影历史、评分推荐 | 用户历史订单、评分记录 |
| 基于热度 | 推荐当前热门电影 | 票房排行、评分排行 |
| 基于类型 | 用户说"推荐一部喜剧片" | 电影类型筛选 |
| 基于场景 | 用户说"适合情侣看的电影" | 电影标签匹配 |

### 3.2 影院推荐

| 推荐场景 | 推荐逻辑 | 数据来源 |
|----------|----------|----------|
| 基于位置 | 推荐距离最近的影院 | 用户位置、影院地址 |
| 基于评分 | 推荐评分最高的影院 | 影院评分数据 |
| 基于设施 | 推荐有 IMAX/Dolby 的影院 | 影院设施标签 |

### 3.3 优惠券推荐

| 推荐场景 | 推荐逻辑 | 数据来源 |
|----------|----------|----------|
| 即将过期 | 推荐用户即将过期的优惠券 | 用户优惠券列表 |
| 满足条件 | 用户购票时推荐可用优惠券 | 订单金额、优惠券规则 |
| 个性化 | 根据用户偏好推荐适合的券 | 用户画像、优惠券标签 |

## 4. Function Call 能力

### 4.1 智能购票

**用户意图示例：**
- "帮我买一张最近的科幻片"
- "帮我买明天下午《流浪地球3》的票"
- "帮我订两张万达影院晚上的票"

**执行流程：**

```
用户输入 → 意图识别 → 参数提取 → 调用接口 → 结果反馈
    │
    ▼
"帮我买一张最近的科幻片"
    │
    ▼
意图：购票
参数：{ genre: "科幻", quantity: 1, time: "最近" }
    │
    ▼
1. 调用电影列表接口 → 筛选科幻片
2. 调用排片接口 → 查询场次
3. 返回候选列表让用户确认
4. 用户选择场次、座位
5. 调用下单接口 → 完成购票
```

### 4.2 智能退票

**用户意图示例：**
- "帮我退掉明天的票"
- "我想退《流浪地球3》的票"

**执行流程：**

```
用户输入 → 意图识别 → 参数提取 → 调用接口 → 结果反馈
    │
    ▼
"帮我退掉明天的票"
    │
    ▼
意图：退票
参数：{ date: "明天" }
    │
    ▼
1. 调用订单查询接口 → 找到明天订单
2. 校验退票规则（开场前2小时）
3. 返回可退订单让用户确认
4. 调用退票接口 → 完成退票
```

### 4.3 查询订单

**用户意图示例：**
- "查一下我的订单"
- "我在万达影院的消费记录"

**执行流程：**

```
用户输入 → 意图识别 → 参数提取 → 调用接口 → 结果反馈
    │
    ▼
"查一下我在万达影院的消费记录"
    │
    ▼
意图：查询订单
参数：{ cinema: "万达影院" }
    │
    ▼
1. 调用订单查询接口 → 筛选影院
2. 返回订单列表
```

### 4.4 影院推荐（带查询）

**用户意图示例：**
- "帮我推荐附近评分最高的影院"
- "找个有 IMAX 的电影院"

**执行流程：**

```
用户输入 → 意图识别 → 参数提取 → 调用接口 → 结果反馈
    │
    ▼
"帮我推荐附近评分最高的影院"
    │
    ▼
意图：影院推荐
参数：{ location: "附近", sortBy: "评分" }
    │
    ▼
1. 调用影院列表接口 → 按距离筛选
2. 按评分排序
3. 返回推荐结果
```

## 5. Function Call 接口清单

### 5.1 需要调用的主服务接口

| 接口 | 用途 | 对应 Function |
|------|------|---------------|
| `/api/movies` | 查询电影列表 | 电影推荐、智能购票 |
| `/api/cinemas` | 查询影院列表 | 影院推荐 |
| `/api/schedules` | 查询排片信息 | 智能购票 |
| `/api/orders` | 查询/创建订单 | 智能购票、查询订单、智能退票 |
| `/api/orders/{id}/refund` | 申请退票 | 智能退票 |
| `/api/coupons` | 查询优惠券 | 优惠券推荐 |
| `/api/users/{id}/history` | 查询用户历史 | 个性化推荐 |

## 6. 对话交互设计

### 6.1 多轮对话示例

**场景：智能购票**

```
用户：帮我买一张科幻片
AI：好的，为您推荐以下科幻片：
    1. 流浪地球3 - 评分 9.2
    2. 三体 - 评分 8.8
    请问您想看哪一部？

用户：流浪地球3
AI：好的，《流浪地球3》，请问您在哪个影院观影？
    为您推荐附近影院：
    1. 万达影城（XX店）- 距离 1.2km
    2. CGV 影城（XX店）- 距离 2.5km

用户：万达影城
AI：好的，万达影城今日《流浪地球3》排片：
    1. 14:30 - IMAX厅
    2. 17:00 - 普通厅
    3. 19:30 - 杜比厅
    请选择场次。

用户：19:30
AI：好的，19:30 杜比厅，请问您需要选择哪个座位？
    [座位图展示]

用户：F排8座
AI：已为您选择 F排8座，票价 45 元。
    确认下单吗？

用户：确认
AI：下单成功！订单号：202505020001
    取票码：886655，请凭此码到影院取票。
```

### 6.2 错误处理

| 场景 | AI 回复 |
|------|---------|
| 无匹配电影 | "抱歉，没有找到相关电影，换个说法试试？" |
| 无场次 | "该影院暂无此电影排片，建议选择其他影院" |
| 座位已售 | "该座位已被选购，请重新选择" |
| 退票失败 | "该订单已开场，无法退票" |

## 7. 优先级

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 智能客服 - 业务咨询 | 高 | 基础能力 |
| 智能客服 - 规则解释 | 高 | 基础能力 |
| Function Call - 智能购票 | 高 | 核心亮点 |
| Function Call - 查询订单 | 高 | 常用功能 |
| Function Call - 智能退票 | 中 | 需校验规则 |
| 智能推荐 - 电影推荐 | 中 | 需用户画像 |
| 智能推荐 - 影院推荐 | 中 | 需位置信息 |
| 智能推荐 - 优惠券推荐 | 低 | 可简化 |
