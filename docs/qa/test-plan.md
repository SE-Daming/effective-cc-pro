# 测试计划

> 仿猫眼电影票务平台 - 后端与B端管理后台测试计划

---

## 1. 测试概述

### 1.1 测试目标

验证后端 API 和 B 端管理后台的功能正确性，发现并修复 Bug。

### 1.2 测试范围

| 范围 | 说明 |
|------|------|
| 后端 API | 所有 RESTful 接口 |
| B 端前端 | 管理后台所有页面功能 |
| 排除 | C 端小程序（已单独测试）、AI 助手（未开发） |

### 1.3 测试环境

| 环境 | 地址 | 说明 |
|------|------|------|
| 后端服务 | http://localhost:8080/api | Spring Boot |
| API 文档 | http://localhost:8080/api/doc.html | Knife4j |
| 前端服务 | http://localhost:3000 | Vue 3 |
| 数据库 | localhost:3306/maoyan | MySQL |
| Redis | localhost:6379 | 可选 |

## 4. 测试执行顺序

5. 测试排片管理

6. 测试订单管理

10. 测试影评管理
