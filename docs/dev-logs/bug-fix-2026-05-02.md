### B002 - 分类列表接口返回 500 错误

**状态**: ✅ 已修复

**原因分析**:
- 前端请求路径: `/admin/categories`
- 后端接口路径: `/admin/movie-categories`
- 路径不匹配导致 404，被全局异常处理捕获返回 500

**修复方案**:
1. 修改前端 API 路径 `frontend/web/src/api/movie.js`:
   ```javascript
   // 修改前
   url: '/admin/categories'
   // 修改后
   url: '/admin/movie-categories'
   ```

### B012 - 电影编辑页面显示"服务器内部错误"

**状态**: ✅ 已修复

**原因分析**:
1. 缺少获取电影详情的管理端接口 `GET /admin/movies/{id}`
2. 状态值不一致：后端 1/2/3 对应前端 1/2/0
3. 分类选项硬编码，未动态获取

**修复方案**:

1. **后端添加电影详情接口**:
   - `AdminMovieController.java` 添加 `getMovieDetail` 方法
   - `MovieService.java` 添加 `getMovieDetailAdmin` 方法签名
   - `MovieServiceImpl.java` 实现 `getMovieDetailAdmin` 方法

2. **后端添加品牌管理接口**:
   - 创建 `AdminCinemaBrandController.java`
   - 路径: `/admin/brands`

3. **前端修复状态值** `frontend/web/src/views/movie/edit.vue`:
   ```javascript
   // 修改前
   <el-radio :value="0">已下架</el-radio>
   // 修改后
   <el-radio :value="3">已下架</el-radio>
   ```

4. **前端动态获取分类**:
   - 添加 `getMovieCategories` API 调用
   - 分类选择器改为动态渲染
   - 处理 `categoryIds` 字符串与数组的转换
