## 2025-05-02 小程序搜索模块开发

### 开发内容

1. **搜索 API 接口** (`api/search.js`)
   - 搜索电影接口（复用 movie.js）
   - 搜索影院接口（复用 cinema.js）
   - 获取搜索历史接口
   - 清空搜索历史接口
   - 热门搜索关键词（本地数据）

2. **搜索页面** (`pages/search/search.vue`)
   - 搜索框自动聚焦
   - 搜索类型切换（电影/影院）
   - 搜索历史展示
   - 热门搜索推荐
   - 搜索结果列表
   - 搜索防抖处理（500ms）
   - 本地历史存储

3. **路由配置**
   - 在 pages.json 中添加搜索页面路由

### 功能特点

- 支持点击历史/热门关键词直接搜索
- 搜索历史本地存储，失败时使用本地历史
- 搜索结果支持加载更多
- 空状态和加载状态展示
