## 3. 接口路径差异说明

测试过程中发现测试计划文档部分接口与实际实现存在差异，汇总如下：

| 测试计划路径 | 实际实现路径 | 说明 |
|--------------|--------------|------|
| PUT /admin/reviews/{id}/audit | PATCH /admin/reviews/{id}/audit | HTTP方法不同 |
| PUT /admin/users/{id}/status | PATCH /admin/users/{id}/status | HTTP方法不同 |
| PUT /admin/orders/{id}/refund | POST /admin/orders/{id}/refund-audit | 路径和方法都不同 |
| POST /admin/coupons/{id}/issue | POST /admin/coupons/{id}/distribute | 路径不同 |
| GET /admin/movies/categories | GET /admin/movie-categories | 路径不同 |
| GET /admin/cinemas/{id}/halls | GET /admin/halls?cinemaId=xx | 路径结构不同 |
| POST /admin/cinemas/{id}/halls | POST /admin/halls | 路径结构不同 |

建议后续更新测试计划文档，使接口路径与实际实现保持一致。
